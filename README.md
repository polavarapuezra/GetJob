GetJob Portal is a full-stack web application designed to help users discover, track, and manage job opportunities in a structured and efficient way. The platform aggregates job listings from multiple sources and presents them through a clean, skill-based filtering interface.

Users can browse jobs based on their skills, view match percentages, and directly navigate to external job portals to apply. The system allows users to mark jobs as “Applied,” enabling basic tracking of their application activity and progress.

The application is built using a Flask backend with dynamic HTML templates and integrates a database to store user data, applied jobs, and personalized skill sets. The interface emphasizes usability, providing real-time feedback through interactive UI elements such as dynamic apply buttons and status indicators.

Deployed on a cloud platform, the application is accessible via a public domain, allowing real users to interact with the system in a live environment.

🔧 Key Features
Multi-portal job listing system
Skill-based job filtering and match percentage display
One-click external job application redirection
Application tracking with “Applied” status
User-specific skill management
Responsive UI with dynamic button state handling
Cloud deployment with public access
🧰 Tech Stack
Frontend: HTML, CSS, Bootstrap, JavaScript
Backend: Python (Flask)
Database: SQL (SQLite / upgraded support for scalable DB)
Deployment: Render
Version Control: GitHub
